# Prime esercitazioni AWS

Questo repository contiene le mie prime attività su Amazon Web Services.

## Servizi usati

### S3 (Simple Storage Service)
- Creato un bucket
- Caricato un file
- Capito il concetto di storage cloud

### EC2 (Elastic Compute Cloud)
- Creata una istanza Amazon Linux
- Capito cos'è un computer su internet
- Arrestata l'istanza per evitare costi

## Obiettivo
Imparare le basi del cloud AWS passo dopo passo.
