# S3 - Simple Storage Service

In questa esercitazione ho usato Amazon S3 come storage cloud.

## Cosa ho fatto
- Creato un bucket
- Caricato un file dal mio PC
- Capito il concetto di storage su cloud

## Cosa ho imparato

# EC2 - Elastic Compute Cloud

In questa esercitazione ho creato una istanza EC2.

## Cosa ho fatto
- Creato un server Amazon Linux
- Usato un'istanza Free Tier
- Arrestato il server per evitare costi

## Cosa ho imparato
- EC2 è un computer su internet
- I server si possono accendere e spegnere
- Si paga solo quando il server è acceso
